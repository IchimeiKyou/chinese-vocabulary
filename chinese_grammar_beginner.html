Unit 19 对照 HSK 1 官方 54 语法点补全。每个例句拆到「这个词是什么意思、在句子里干什么」。

📖 怎么读文法单元？

先读导读建立框架，再看黄色规则框，然后重点看绿色逐词拆解——把日语 SOV 思维翻成中文 SVO。蓝色「日语对比」和红色「注意」是日本人专属坑点；豆知识帮你记为什么这样说不那样说。每单元结束试造 3 句自己的话。

Unit 1 声调与拼音声调与拼音
中国语有四个声调加轻声。日语没有「同一个音、声调不同意思就不同」的系统，所以要单独练。
📖 声调：中国语的「旋律」

同一音节声调不同=不同词。日语无此系统，声调须单独练。

🫘 第三声的秘密

第三声连续说话时 often 只降不升——先掌握四声对比，再追求语速。

拼音就像中国语的「发音表」。每个字用拉丁字母标音，再标声调。先记住：看拼音读，不要只看汉字按日语念。

拼音怎么拆

一个音节 = 声母（开头的辅音）+ 韵母（后面的元音等）+ 声调。例：mā（妈）= m + a + 第一声。

四声对照（初学者版）

声调符号例日语学习者提示
第一声āmā 妈（母亲）高而平，像拉长说「あ——」
第二声ámá 麻从中往高升，像说「え？」
第三声ǎmǎ 马先降下去再升起来（最难，多练）
第四声àmà 骂从高快速落下，像短促的「あっ」
轻声mama 吗（疑问）短而轻，不标调号
妈妈骑马。 Māma qí mǎ.
→ お母さんが馬に乗る。
词读音意思（初学者向け）
妈māお母さん（第一声・高平）
妈māまた「妈」— 妈＋妈＝妈妈
骑qí乗る（第二声・上がる）
马mǎ馬（第三声・降って上がる）
句型：主语 + 动词 + 宾语（和 Unit 3 一样）
四声组 mā/má/mǎ/mà 建议背下来。第三声日本人最容易读成低平调。
你好。 Nǐ hǎo.
→ こんにちは／お元気ですか。
词读音意思（初学者向け）
你nǐあなた（第二声）
好hǎo良い・元気（第三声）
句型：问候语：你 + 好
声调错了听起来会像别的词。先慢读，再加快。
🇯🇵 日语对比

日语没有靠声调区分词义。中国人靠声调听出是哪个字。

不要把日语的长短音习惯直接套到声调上；第三声要刻意练「先降后升」。

⚠️ 常见错误

只看汉字用日语音读念——发音会全错。务必对照拼音。

Unit 2 汉字与简繁汉字与简繁
大陆用简体字，日本用新字体。字形有时一样，读音往往完全不同。
🫘 简体字怎么来的？

1950年代大陆推简体提高识字率。日本用新字体，台湾用繁体。

汉字能帮你记意思，但不能帮你记读音。初学者策略：认字形 + 记拼音，两套一起背。

第一步记哪些字

先认超高频字：我、你、他、是、不、好、人、大、小。看到能认出意思即可，读音跟拼音走。

我是中国人。 Wǒ shì Zhōngguó rén.
→ 私は中国人です。
词读音意思（初学者向け）
我wǒ私（第三声）
是shì〜です（第四声）
中国Zhōngguó中国（国名）
人rén人（第二声）
句型：我 + 是 + 名词
「中国」两个字：中＝真ん中、国＝国。日本語の「中国」と同じ漢字。
他学中文。 Tā xué Zhōngwén.
→ 彼は中国語を学ぶ。
词读音意思（初学者向け）
他tā彼（第一声）
学xué学ぶ（第二声）
中文Zhōngwén中国語（文＝言語・文字）
句型：主语 + 动词 + 宾语
「学习」在日本是がくしゅう，在中国是 xuéxí——同形异音の典型例。

同形字读音对比

汉字中国语读音日语读音提示
学习xuéxíがくしゅう字形同，音完全不同
电话diànhuàでんわ電話——音も違う

⚠️ 最大陷阱

看到汉字就按日语读——这是日语母语者第一大坑。必须拼音 + 汉字一起记。

Unit 3 SVO 基本语序SVO 基本语序
中文基本顺序：主语 → 动词 → 宾语（SVO）。日语是 SOV，动词在最后——这是最大的习惯差异。
📖 SOV→SVO 思维转换

三框：【谁】【做什么】【什么】。中文动词放中间框。

🫘 语序对比

SVO=中/英；SOV=日/韩。日语母语者需多一步思维翻转。

造句时先在脑子里排顺序：谁 + 做什么 + 什么。不要按日语「谁 + 什么 + 做」来排。

基本公式（背这个）

主语 + 动词 + 宾语。例：我（谁）+ 看（做什么）+ 书（什么）。

我吃饭。 Wǒ chī fàn.
→ 私はご飯を食べる。
词读音意思（初学者向け）
我wǒ私（話し手）
吃chī食べる
饭fànご飯・食事（饭＝炊いた米、日常は「食事」）
句型：主语 + 动词 + 宾语（SVO）
日语：私はご飯を食べる（SOV）。中文：我吃饭（SVO）。动词「吃」在饭「饭」前面。
小狗喝水。 Xiǎo gǒu hē shuǐ.
→ 子犬が水を飲む。
词读音意思（初学者向け）
小xiǎo小さい
狗gǒu犬
喝hē飲む
水shuǐ水
句型：（形容词）+ 名词 + 动词 + 宾语
小狗＝小＋狗。没有助词「の」，直接连在一起。
🇯🇵 日语对比

日语：私は本を読む（SOV）——动词在最后。

中文：我看书（SVO）——动词在中间。先练把动词放到宾语前面。

⚠️ 语序错误

不要说「书看」「饭吃」。强制练习：我 + 动词 + 宾语。

Unit 4 「是」字句「是」字句
「是」= 日语的「だ／です」，连接名词。说「我是学生」时用；说「我很高兴」时不用。
🫘 「是」不能连形容词

我是学生○，我是高兴×→我很高兴。日语「です」两者都能连。

判断「A 是 B」（A＝名词、B＝名词）→ 中间加「是」。形容词作谓语时不用「是」。

肯定与否定

肯定：主语 + 是 + 名词。否定：主语 + 不是 + 名词。例：我是老师／我不是老师。

我是学生。 Wǒ shì xuésheng.
→ 私は学生です。
词读音意思（初学者向け）
我wǒ私
是shì〜です（名词判断）
学生xuésheng学生（学＝学ぶ、生＝生まれる→学生）
句型：主语 + 是 + 名词
回答「你是不是学生？」→ 是／不是（不用はい／いいえ，要说「是」或「不是」）。
这不是书。 Zhè bú shì shū.
→ これは本ではありません。
词读音意思（初学者向け）
这zhèこれ
不bú〜ない（否定）
是shì〜です
书shū本
句型：指示词 + 不是 + 名词
「不」在「是」前面：不是＝ではない。
⚠️ × 我是好。 ○ 我很好。

形容词（好、大、冷）前面不加「是」。要说「我很好」而不是「我是好」。

🇯🇵 日语对比

「是」只用于名词判断，不是万能 copula。

日语的「です」什么都能接；中文要分「是」和「很+形容词」两套。

Unit 5 否定与疑问否定与疑问
否定：在动词或形容词前加「不」。一般疑问：陈述句末尾加「吗」。
📖 否定与疑问

否定加「不」；一般疑问句末加「吗」；特殊疑问用疑问词。

先学两种否定：「不」= 现在／将来不做；「没」= 过去没做（后面 Unit 会细讲）。疑问句先学加「吗」。

否定怎么加

不 + 动词：不去、不吃。不 + 形容词：不好、不贵。放在动词／形容词前面。

一般疑问

陈述句 + 吗：你去吗？= 行きますか。句末语气变疑问，不用日语的「か」。

我不去。 Wǒ bú qù.
→ （私は）行きません。
词读音意思（初学者向け）
我wǒ私
不bú〜ない（否定）
去qù行く
句型：主语 + 不 + 动词
「不」读第二声 bú（后面跟第四声时变调）。日语没有单独的否定词位置——中文否定在动词前。
你好吗？ Nǐ hǎo ma?
→ お元気ですか。
词读音意思（初学者向け）
你nǐあなた
好hǎo元気・良い
吗ma〜ですか（疑问语气，轻声）
句型：主语 + 形容词 + 吗
「你好吗」= 你好 + 吗。学习阶段建议疑问句都加「吗」，比日语「か」更明确。
你在哪儿？ Nǐ zài nǎr?
→ どこにいますか。
词读音意思（初学者向け）
你nǐあなた
在zài〜にいる・〜にある
哪儿nǎrどこ（疑问词，儿化音）
句型：主语 + 在 + 疑问词
有特殊疑问词（哪儿、谁、什么）时，一般不加「吗」。
🇯🇵 日语对比

日语靠句尾「か」或语调；中文一般疑问要加「吗」，或换疑问词。

「你好吗」口语有时省略「吗」，但初学者建议一律加上。

Unit 6 形容词谓语句形容词谓语句
形容词可以直接当谓语（说「天气冷」），但通常加「很」：天气很冷。
🫘 「很」≠ very

「很高兴」的「很」often 只是语法需要；强调用「非常／太」。

中文形容词不像日语要变成「〜い／〜だ」——直接放谓语位置。但单独说「我好」不自然，要加「很」。

基本结构

主语 + （很）+ 形容词。否定：主语 + 不 + 形容词。例：他很高／他不高。

今天很冷。 Jīntiān hěn lěng.
→ 今日はとても寒い。
词读音意思（初学者向け）
今天jīntiān今日（今天＝今＋天）
很hěnとても（程度，但不总是 very）
冷lěng寒い
句型：时间 + 很 + 形容词
时间词放前面，形容词谓语后面不需要「是」。
我很好。 Wǒ hěn hǎo.
→ （お返事）元気です。
词读音意思（初学者向け）
我wǒ私
很hěn（ここでは程度強調ではない）
好hǎo良い・元気
句型：主语 + 很 + 形容词
回答「你好吗？」时说「我很好」——「很」在这里不表示「とても」，只是习惯用法。
这个不贵。 Zhège bú guì.
→ これは高くない。
词读音意思（初学者向け）
这个zhè geこれ（这＋个）
不bú〜ない
贵guì高い（値段が）
句型：指示 + 不 + 形容词
否定形容词：不 + 贵。不用「没」。
⚠️ 「很」不一定是 very

「我很好」常只是礼貌回应「元気です」，不强调「とても良い」。

Unit 7 数字与量词数字与量词
中文数东西：数字 + 量词 + 名词。不能直接说「三书」，要说「三本书」。
🫘 「两」和「二」

量词前用「两」；数数字、第几用「二」。

量词类似日语助数词（一枚、一匹），但种类更多。先记最常用的几个，见到名词再想用哪个。

公式（三步）

① 想数字 ② 选量词 ③ 加名词。例：三 + 本 + 书 = 三本书。

「二」的特殊用法

量词前面说「两」，不说「二」：两本书、两个人。数数字本身仍用二、三、四…

两本书。 Liǎng běn shū.
→ 本が2冊。
词读音意思（初学者向け）
两liǎng2（量词前用「两」）
本běn冊（书的量词）
书shū本
句型：数词 + 量词 + 名词
本＝书的专用量词。日本人说「二冊」用助数词；中文把「本」嵌在数字和名词之间。
一个人。 Yí gè rén.
→ 一人。
词读音意思（初学者向け）
一yí1（后面跟四声时读 yí）
个gè个＝通用量词（人、苹果等）
人rén人
句型：数词 + 量词 + 名词
「个」是最万能的量词，不知道用什么时初学者可先想「个」（不总是对，但入门够用）。

常用量词入门表

🇯🇵 日语对比

和日语助数词（一枚、一匹）类似，但要逐个记。

量词前数字「2」用「两」：两本书，不说二本书。

Unit 8 代词与指示代词与指示
人称代词没有性别变化（他＝男も女も）。「这／那」要加量词才像日语「これ／あれ」。
📖 指示系统

这=近、那=远、哪=疑问——须加量词：这个、那个。

「这」单独不够，要说「这个」「这些」。「的」表示所属，类似日语「の」。

「的」怎么用

名词 + 的 + 名词 = 所属：我的书、老师的电话。位置跟日语「の」一样：我的书＝私の本。

这个很好。 Zhège hěn hǎo.
→ これはとてもいい。
词读音意思（初学者向け）
这个zhè geこれ（这＝この、个＝个）
很hěnとても
好hǎo良い
句型：指示代词 + 很 + 形容词
这个＝这 + 个。日语「これ」一个词；中文「这」+ 量词「个」。
那是我的。 Nà shì wǒ de.
→ あれは私のもの。
词读音意思（初学者向け）
那nàあれ・それ
是shì〜です

句型：那 + 是 + 名词 + 的
句末「我的」= 我的东西。名词「东西」可以省略。
这些是什么？ Zhèxiē shì shénme?
→ これらは何ですか。
词读音意思（初学者向け）
这些zhèxiēこれら（些＝複数）

句型：指示（复数）+ 是 + 疑问词
特殊疑问句：用「什么」提问，不加「吗」。
🇯🇵 日语对比

日语「の」→ 中文「的」；位置相同。

不能说「我 book」——要么「我的书」，要么在上下文里省略。

Unit 9 程度与频率副词程度与频率副词
副词放在动词或形容词前面。「都」「也」「还」位置固定，不能乱放。
🫘 「都」的位置

主语+都+动词：我们都去了。不能说「都我们去」。

副词修饰后面的动词或形容词。先记住：也＝も、都＝みんな、常常＝よく。

「都」放哪里

主语 + 都 + 动词：我们都喜欢。都放在主语后面、动词前面。

「也」放哪里

主语 + 也 + 动词：我也去。表示「同样」。

我也去。 Wǒ yě qù.
→ 私も行く。
词读音意思（初学者向け）
我wǒ私
也yěも（同样）
去qù行く
句型：主语 + 也 + 动词
「也」在动词前，不在句末。日语「も」位置较自由；中文「也」紧贴动词前。
我们都去了。 Wǒmen dōu qù le.
→ 私たちはみんな行った。
词读音意思（初学者向け）
我们wǒmen私たち（们＝複数）
都dōuみんな・すべて
去qù行く

句型：主语 + 都 + 动词 + 了
「了」在句末表示动作完成或情况变化（详见 Unit 13）。
他常常迟到。 Tā chángcháng chídào.
→ 彼はよく遅刻する。
词读音意思（初学者向け）

常常＝频率副词，放在动词短语前面。
⚠️ 「都」别乱放

× 都我们去 → ○ 我们都去。「都」要紧挨动词前，在主语后。

Unit 10 时间表达时间表达
时间词一般放在主语后面、动词前面：我明天去。
📖 时间词位置

时间词放动词前或句首：我今天去。不像日语必须句末。

说「什么时候做」→ 把时间词插在主语和动词之间。这是中文和日语位置差异很大的地方。

时间放哪里

主语 + 时间 + 动词 + 宾语。例：我昨天看了电影 = 私は昨日映画を見た。

→ 私は昨日北京に行った。

Unit 13 了·过·着了·过·着
三个常用「助词」：了（完成／变化）、过（曾经）、着（持续状态）。不要和日语一一硬套。
📖 了/过/着

「我来了」=刚到，不是「来过了」。

「过」= 曾经

「着」= 持续

Unit 16 「把」字句入门「把」字句入门

📖 把字句入门

🫘 何时用「把」？

宾语被处置时用：把饭吃了、把门关上。单纯「我看书」不用把。
